# Samples for Groovy Training

## Setup

### Tools
* install GGTS or Eclipse with Groovy Plugin

### Setup project
* clone repository
* import project

### Database
* database sample uses hsqldb
* start scripts (server, console) provided in folder db
* setup database using application.sql
